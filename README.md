# TxtReader

 
## 一 、这是一个Txt文本阅读器，包含翻页效果
 
## 二 、包含功能:
1.翻页效果（贝塞尔曲线）<br> 
2.快进快退效果（百分比值进行）<br> 
3.字体大小<br> 
4.书签效果<br> 

 
## 三、使用方法：
 
1.gradle 中添加：    compile 'com.yansheng.txtreaderplayer:ebookread:1.1'，<br> 
2.调用ReadActivity.actionStart(context,"电子书标题","电子书路径")。<br>
